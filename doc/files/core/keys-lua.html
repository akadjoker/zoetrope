Keys

This tracks the state of the keyboard, i.e. if a key was just pressed or released this frame.  You can look up a key either by its name or its Unicode number.  Not all keys sensed have Unicode equivalents (e.g. modifiers like control or alt).

Only one Keys object can be active at one time.  The one currently listening to the keyboard can be accessed globally via the.keys.

See http://love2d.org/wiki/KeyConstant for a list of key names.  This class aliases modifiers for you, so that if you want to check whether either the left or right Control key is pressed, you can check on ‘ctrl’ instead of both ‘lctrl’ and ‘rctrl’.
