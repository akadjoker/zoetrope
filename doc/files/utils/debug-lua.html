DebugConsole

A debug console displays the value of an expression each frame.  It can be used to keep track of fps, the position of a sprite, and so on.  It only updates when visible.

This also allows debugging hotkeys -- e.g. you could set it so that pressing Control-Alt-I toggles invincibility of the player sprite.  Out of the box, Control-Alt-R reloads all app code from on disk, Control-Alt-Q quits the app, and Control-Alt-F toggles fullscreen.

Properties

toggleKey

What key toggles visibility.  By default, this is the tilde key.

hotkeyModifiers

A table of modifier keys that must be held in order to activate a debugging hotkey (set via <bind()>).  If you want hotkeys to activate without having to hold any keys down, set this to nil.

initWithFPS

If true, the watch will automatically start watching the frames per second.  Changing this value after the DebugWatch object has been created has no effect.

watchWidth

How wide the sidebar, where watch values are displaed, should be.
