Cached

This helps you re-use assets in your app instead of creating extraneous copies of them.  It also hides Love-related calls so that your code is more portable.

This class is not meant to be created directly.  Instead, call methods on Cached directly, e.g.  Cached:sound, Cached:image, and so on.

sound

Returns a cached sound asset.  Be careful using this method.  A single sound can only be played once, so if you need to be able to play the same sound simultaneously, you’ll need to set up separate entries for it via the alias property.

Really, you are probably better served by using the Jukebox class to manage your sounds for you if you are worried about multiple simultaneous sounds.
