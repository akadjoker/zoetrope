Jukebox

This helps manage playing one-off sounds.  It’s great if you don’t need to do anything to a sound except play it.

play

Plays a sound.

Arguments

  • source - either a Sound object or string path.  If you use a string path, then repeated play() calls will cause simultaneous sounds to play.  If you pass a Sound object directly, it will cause the sound to restart.
  • volume - volume, from 0 to 1

Returns

Sound object that just began playing
