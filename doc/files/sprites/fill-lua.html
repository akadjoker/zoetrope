Properties

fill

A table of color values in RGBA order.  Each value should fall between 0 and 255.  The fill sprite fills this color in its bounds onscreen.

border

A table of color values in RGBA order.  Each value should fall between 0 and 255.  The fill sprite draws the border in this color after filling in a color (if any).
